# Asymptote-Calculator
Computes asymptotes

# Dependencies
Sympy v1.1.1+

# Usage
**Commandline:**
```
python "path\asymptote calculator.py" "equation"
```
**Python Shell**
```
Equation
```

# Examples
**Commandline:**
```
>python "path\asymptote calculator.py" "2^x+2"
Asymptote:
y = 2
```
**Python Shell:**
```
Equation: y = (2*x^2+3*x+1)/x
Asymptote:
y = 2*x + 3
```
